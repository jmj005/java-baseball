1. 컴퓨터 랜덤 3자리 수 생성


2. 콘솔 게임 시작 출력
   

3. 사용자 3자리 수 입력  
   - 입력 값 확인  
        - 범위가 맞는지 확인  
        - 3자리 수가 맞는지 확인
   -  잘못됐다면 에러 발생 후 종료
   

4. 낫싱 판단  
   - 낫싱이면 7번 이동 후 3번으로 이동
    

5. 볼 판단 
   - 6번 이동
    

6. 스트라이크 판단  
   - 3스트라이크면 8로 이동
   - 아니라면 7번 이동 후 3번으로 이동
   

7. 콘솔 게임 결과 출력
   

8. 게임 종료 조건이라면  
    - 콘솔 게임 종료 출력  
    - 입력 값을 받음  
        - 1을 입력하면 새로 시작
            - 1번으로 이동
        - 2를 입력하면 게임 종료
